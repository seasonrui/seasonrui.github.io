链接

硬链接：每个文件都有一个硬链接，硬链接和文件没什么区别。
软链接（符号链接）：类似 window 的快捷方式，软链接可以关联一个目录，硬链接不行。

创建链接 ln
-s 表示创建软链

查看文件路径 ls -l
如果文件或目录是软链，-> 后面是链接的真实路径
